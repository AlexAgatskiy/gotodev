# Gotodev - Сборник задач по Го

В этом репозитории собраны платформенные задачи на усвоение разных тем в Го:
- Горутины/каналы
- Слайсы
- Другие (структуры в Го, интерфейсы, ...)

На них можно понять, как работает Го на практике, отработать подводные камни
языка и набить руку на решении задач.

### Как взаимодействовать с репозиторием

#### Процесс решения

1. Решать от легкого уровня к сложному (для той или иной темы есть ранжирование
по уровням сложности задач).
2. Не стоит делать ВСЕ задачи за раз - это бесполезно и займет много времени.
3. Стоит выделить определенные дни в недели, чтобы возвращаться к прорешиванию
снова и снова. Задать себе комфортную планку по кол-ву решенных задач или
затраченному времени.
4. Миксовать разные темы (2 задачи на горутины, 1 по слайсам...)

#### Отправка решений на проверку

1. Создать форк репозитория.
2. Писать решение задач inline (в тех файлах, где они объявлены).
3. После итерации решения создать ПР из своего форка (любая ветка) в этот
репозиторий (main ветка).
4. Через какой-то гэп времени решение будет проверено, и написан фидбек по
нему. Возможно будут дополнительные вопросы на понимание.

#### Директория other

В нее мы смотрим с осторожностью - пока не пройдены все темы по Го, там могут
встретиться непонятные задачки.
